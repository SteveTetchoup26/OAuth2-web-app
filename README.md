# OAuth2-web-app
Secure authentication and authorization system built with spring boot OAuth2. This project demonstrates how to integrate Google login

## Features
This features are available in this project

- **Google login:** Login with your google account

## Prerequisites
- Maven

## Installation
- Clone the repository
```bash
$ git clone https://github.com/SteveTetchoup26/OAuth2-web-app.git [project_name]
```
- Start Application
```bash
$ cd [project_name]
$ npm run dev
```

## Docker
To run the project with docker, just run
```bash
$ docker-compose up --build
```
